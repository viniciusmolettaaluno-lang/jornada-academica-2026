Estruturas Condicionais em Python

🐍 Python 🔄 if/elif/else 📚 Lógica ⚡ Engenharia

Aplicação de estruturas condicionais para resolver problemas de engenharia, com ênfase em tomada de decisão e validação de dados.

🎯 Objetivo

Desenvolver a capacidade de aplicar estruturas condicionais (if, elif, else) em problemas práticos de engenharia, incluindo:

  • Verificação de limites operacionais
  • Seleção de materiais com base em propriedades
  • Validação de dados de sensores
  • Dimensionamento básico

📋 Exemplos de Problemas

1 Verificação de Tensão

Ler um valor de tensão (V) e classificar como "Baixa", "Normal" ou "Alta" conforme faixas pré-definidas.

2 Seleção de Material

Com base na temperatura de operação, sugerir um material adequado (cobre, alumínio, aço).

💻 Código Desenvolvido

# Trabalho 2 - Estruturas Condicionais
# Classificação de tensão elétrica

def classificar_tensao(tensao):
    """Classifica a tensão em faixas operacionais"""
    if tensao < 0:
        return "⚠️ Valor inválido (negativo)"
    elif tensao < 110:
        return "🔵 Tensão Baixa"
    elif tensao <= 220:
        return "🟢 Tensão Normal"
    elif tensao <= 380:
        return "🟡 Tensão Elevada"
    else:
        return "🔴 Tensão Crítica"

# Testando a função
valores_teste = [50, 127, 220, 300, 440, -10]
for v in valores_teste:
    print(f"Tensão {v}V → {classificar_tensao(v)}")
▶ Saída do programa
Tensão 50V → 🔵 Tensão Baixa Tensão 127V → 🟢 Tensão Normal Tensão 220V → 🟢 Tensão Normal Tensão 300V → 🟡 Tensão Elevada Tensão 440V → 🔴 Tensão Crítica Tensão -10V → ⚠️ Valor inválido (negativo)

💡 Conceitos aplicados: Estruturas condicionais aninhadas, validação de entrada, uso de múltiplos elif para faixas de valores, aplicação em contexto de engenharia elétrica.
