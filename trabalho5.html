Listas / Vetores / Arrays em Python

🐍 Python 📋 Listas ⚡ Engenharia Elétrica 📊 Análise de Dados

Implementação de 15 exercícios práticos utilizando listas/vetores/arrays em Python, aplicados a problemas de engenharia elétrica, telecomunicações e análise de dados.

🎯 Objetivo do Trabalho

Escrever o código Python dos exercícios de listas/vetores/arrays. Para cada exercício, implementar e executar o código Python, mostrando a execução na máquina.

  • Manipulação de listas (append, remove, extend, copy)
  • Operações com listas (sum, max, min, len, sort)
  • Filtragem e busca em listas
  • Aplicações em engenharia (temperaturas, tensões, pacotes, cargas, sinais)

💻 Código Exemplo (Exercício 5 - Cargas Elétricas)

# Exercício 5 — Cargas Elétricas
# Cálculo de potência ativa após remoção de cargas

cargas = [15.0, 22.5, 8.0, 40.0, 12.5, 30.0]

potencia_antes = sum(cargas)

cargas.remove(8.0) # Desligada por sobrecarga
cargas.remove(40.0) # Equipamento em manutenção

potencia_agora = sum(cargas)
reducao = potencia_antes - potencia_agora

print(f"Cargas ativas : {cargas}")
print(f"Potência antes: {potencia_antes:.1f} kW")
print(f"Potência agora: {potencia_agora:.1f} kW")
print(f"Redução : {reducao:.1f} kW")

# Saída esperada:
# Cargas ativas : [15.0, 22.5, 12.5, 30.0]
# Potência antes: 128.0 kW
# Potência agora: 80.0 kW
# Redução : 48.0 kW
▶ Saída do programa
Cargas ativas : [15.0, 22.5, 12.5, 30.0] Potência antes: 128.0 kW Potência agora: 80.0 kW Redução : 48.0 kW

💻 Código Exemplo (Exercício 12 - Cobertura de Sinal)

# Exercício 12 — Cobertura de Sinal
# Análise de cobertura de 3 antenas (limiar -100 dBm)

antena_norte = [-85, -90, -105, -78, -92, -110, -88]
antena_sul = [-95, -88, -102, -79, -85, -91, -97]
antena_leste = [-80, -75, -108, -93, -86, -103, -82]

limiar = -100

todos_logs = []
todos_logs.extend(antena_norte)
todos_logs.extend(antena_sul)
todos_logs.extend(antena_leste)

total_amostras = len(todos_logs)
cobertura_adequada = [leitura for leitura in todos_logs if leitura >= limiar]
em_cobertura = len(cobertura_adequada)
taxa_cobertura = (em_cobertura / total_amostras) * 100
pior_sinal = min(todos_logs)
indice_pior = todos_logs.index(pior_sinal)
media = sum(todos_logs) / total_amostras

print(f"Total de amostras : {total_amostras}")
print(f"Em cobertura (>= -100): {em_cobertura}")
print(f"Taxa de cobertura : {taxa_cobertura:.1f}%")
print(f"Pior sinal : {pior_sinal} dBm (índice {indice_pior})")
print(f"Média : {media:.1f} dBm")

# Saída esperada:
# Total de amostras : 21
# Em cobertura (>= -100): 16
# Taxa de cobertura : 76.2%
# Pior sinal : -110 dBm (índice 5)
# Média : -91.0 dBm
▶ Saída do programa
Total de amostras : 21 Em cobertura (>= -100): 16 Taxa de cobertura : 76.2% Pior sinal : -110 dBm (índice 5) Média : -91.0 dBm
