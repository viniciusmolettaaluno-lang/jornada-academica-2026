🧪 Demonstração da Solução Python · IA

Abaixo, um trecho do código principal do agroMindpampaAI, mostrando a lógica de predição e a integração com dados de sensores. 

# agroMindpampaAI - Módulo de Predição
import pandas as pd
import numpy as np
from sklearn.ensemble import RandomForestRegressor

# Dados simulados de sensores (temperatura, umidade, precipitação)
dados = {
    'temp': [22, 24, 21, 23, 25],
    'umid': [65, 70, 68, 72, 66],
    'precip': [12, 8, 15, 5, 10],
    'produtividade': [3200, 3450, 3100, 3600, 3300]
}

df = pd.DataFrame(dados)

# Treinando modelo preditivo
X = df[['temp', 'umid', 'precip']]
y = df['produtividade']
modelo = RandomForestRegressor(n_estimators=100, random_state=42)
modelo.fit(X, y)

# Fazendo uma predição para novos dados
novo_dado = np.array([[23, 68, 10]])
predicao = modelo.predict(novo_dado)
print(f"Produtividade estimada: {predicao[0]:.0f} kg/ha")
▶ Saída do programa
Produtividade estimada: 3350 kg/ha

💡 Como foi feito: O código acima foi desenvolvido em conjunto com LLMs, que auxiliaram na estruturação do modelo, escolha de parâmetros e interpretação dos resultados. O modelo Random Forest foi escolhido por sua robustez com dados não-lineares.
