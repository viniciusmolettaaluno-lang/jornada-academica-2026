🎯 Objetivo do Projeto

O projeto AgroMind Pampa AI foi desenvolvido com o objetivo de simular um sistema inteligente de agricultura de precisão voltado ao monitoramento e gerenciamento da irrigação agrícola, resolvendo problemas relacionados ao desperdício de água, consumo excessivo de energia e dificuldade de monitoramento das condições do solo e do clima.

  • Monitoramento de múltiplos setores agrícolas simultaneamente
  • Inteligência artificial preditiva para antecipar estresse hídrico
  • Irrigação preventiva automatizada baseada em índices de risco
  • Dashboard interativo com indicadores operacionais e alertas
  • Estrutura escalável para sensores reais, APIs e Machine Learning

# Índice de estresse hídrico baseado em variáveis climáticas
def calcular_indice_estresse(temperatura, vento, evaporacao, umidade_solo):
    """
    Calcula o índice de estresse hídrico baseado em múltiplas variáveis.
    Quanto maior o índice, maior o risco de estresse.
    """
    indice = (
        (temperatura / 40) * 0.35 +
        (vento / 50) * 0.25 +
        (evaporacao / 10) * 0.20 +
        (1 - umidade_solo / 100) * 0.20
    )
    return indice

# Classificação do risco
def classificar_risco(indice):
    if indice < 0.3:
        return "Baixo"
    elif indice < 0.5:
        return "Moderado"
    elif indice < 0.7:
        return "Alto"
    else:
        return "Crítico"
