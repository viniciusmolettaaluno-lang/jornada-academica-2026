Exemplo Exercício tipo: "Elabore um programa que leia a temperatura em Celsius e informe se está abaixo de 0°C (congelando), entre 0°C e 30°C (temperatura amena) ou acima de 30°C (quente)."

💻 Código Exemplo

# Exercício de estrutura condicional
# Verifica a faixa de temperatura

def classificar_temperatura(temp):
    if temp < 0:
        return "❄️ Congelando"
    elif temp <= 30:
        return "🌤️ Temperatura amena"
    else:
        return "🔥 Quente"

# Teste da função
temperaturas = [-5, 15, 35, 0, 30]
for t in temperaturas:
    print(f"{t}°C → {classificar_temperatura(t)}")
▶ Saída do programa
-5°C → ❄️ Congelando 15°C → 🌤️ Temperatura amena 35°C → 🔥 Quente 0°C → 🌤️ Temperatura amena 30°C → 🌤️ Temperatura amena

💡 Conceitos abordados: Estruturas condicionais aninhadas, operadores de comparação, boas práticas de legibilidade com elif.
