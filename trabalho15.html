🌐 Trabalho 15

Desenvolvimento do Portfólio Acadêmico

🌐 HTML/CSS 📊 JavaScript 🐍 Python 🚀 GitHub Pages

Desenvolvimento completo do Portfólio Acadêmico como ferramenta de documentação e apresentação dos trabalhos realizados na disciplina de Algoritmos e Programação, integrando conhecimentos de programação web, design e organização de conteúdo.

🎯 Objetivo do Trabalho

Desenvolver um site estático (portfólio acadêmico) para documentar, organizar e apresentar todos os trabalhos realizados na disciplina de Algoritmos e Programação. O site foi projetado para ser intuitivo, visualmente agradável e funcional, servindo como uma vitrine do aprendizado ao longo do semestre.

  • Criar uma estrutura de navegação clara e organizada
  • Apresentar cada trabalho de forma detalhada e interativa
  • Utilizar HTML, CSS e JavaScript para construção do front-end
  • Integrar elementos visuais como cards, tags e blocos de código
  • Publicar o site no GitHub Pages para acesso público

📂 Estrutura do Site

📁 Organização dos Arquivos Estrutura

O site foi organizado de forma hierárquica para facilitar a manutenção:

  • index.html - Página inicial com todos os trabalhos em cards
  • trabalho1.html a trabalho14.html - Páginas dedicadas para cada trabalho
  • style.css - Arquivo de estilos global
  • trabalhos/algoritimos/ - Pasta com os PDFs de cada trabalho
  • script.js - Funcionalidades JavaScript (se houver)

🧭 Navegação UX

A navegação foi projetada para ser simples e intuitiva:

  • Navbar fixa - Presente em todas as páginas com link para a página inicial
  • Navegação entre trabalhos - Botões "Anterior" e "Próximo" em cada página
  • Cards na página inicial - Visualização rápida de cada trabalho
  • Links para PDFs - Acesso direto aos documentos de cada trabalho

🎨 Design Visual UI

O design foi desenvolvido com foco em clareza e profissionalismo:

  • Paleta de cores - Azul escuro, cinza e branco com elementos coloridos para destaque
  • Cards interativos - Efeito hover para melhor experiência do usuário
  • Blocos de código - Syntax highlighting e botão "Copiar"
  • Tags visuais - Identificação rápida de tecnologias e conceitos
  • Design responsivo - Adaptação para diferentes tamanhos de tela

💻 Código Exemplo (Estrutura de um Card)

<!-- Exemplo de card da página inicial -->
<div class="card">
    <h3>📝 Trabalho 1</h3>
    <p>
        Resolução de 15 exercícios práticos com estruturas condicionais 
        simples aplicadas à Engenharia.
    </p>
    <div class="tags">
        <span class="tag blue">🐍 Python</span>
        <span class="tag green">🔄 Estruturas Condicionais</span>
    </div>
    <a href="trabalho1.html" class="botao">
        Ver Trabalho
    </a>
</div>

/* Estilo do card (CSS) */
.card {
    background: white;
    border-radius: 12px;
    padding: 24px;
    box-shadow: 0 4px 16px rgba(0,0,0,0.06);
    border-top: 5px solid #2b6cb0;
    transition: transform 0.2s;
}

.card:hover {
    transform: translateY(-4px);
}
▶ Resultado Visual
Um card interativo com: - Título do trabalho - Descrição resumida - Tags coloridas - Botão "Ver Trabalho" - Efeito hover ao passar o mouse

🔧 Desafios e Soluções

Desafio Solução
Manter consistência visual Criação de um CSS global compartilhado entre todas as páginas
Navegação entre trabalhos Implementação de links "Anterior" e "Próximo" em cada página
Exibição de código com cores Uso de CSS para syntax highlighting manual
Botão "Copiar" funcional JavaScript com Clipboard API para copiar o código
Responsividade Uso de media queries para adaptação a diferentes telas
