🏟️ Trabalho 14

Catálogo de Eventos Esportivos de Alegrete-RS

🐍 Python 📊 Dados 🧠 IA 🏛️ Cultura

Aplicação de Inteligência Artificial e programação Python para o levantamento, estruturação e enriquecimento do conteúdo histórico, digital e cultural do município de Alegrete-RS através de um catálogo de eventos esportivos.

🎯 Objetivo do Projeto

Desenvolver uma aplicação em Python capaz de organizar informações sobre eventos esportivos realizados em Alegrete-RS. A solução gera automaticamente arquivos em formatos estruturados (JSON e CSV) e também uma página web (HTML) para facilitar a visualização e consulta dos eventos.

  • Catálogo contendo nome, modalidade, data, local, histórico e imagens dos eventos
  • Geração automática de arquivos JSON e CSV para integração com outros sistemas
  • Página web responsiva para visualização dos eventos
  • Base para futuras integrações com o portal Alegrete.org
  • Contribuição para a divulgação do esporte no município

💻 Código Exemplo (Geração de Catálogo)

# Código para geração do Catálogo de Eventos Esportivos
# Geração automática de JSON, CSV e HTML

import json
import csv

# Catálogo de eventos esportivos de Alegrete
eventos = [
    {
        "nome": "Titans Race",
        "modalidade": "Corrida de Obstáculos",
        "data": "2026",
        "local": "Alegrete - RS",
        "imagem": "https://via.placeholder.com/400x200",
        "site": "https://www.titansrace.com.br",
        "instagram": "@titansrace",
        "historico": "Evento realizado anualmente."
    },
    {
        "nome": "Minerva Run",
        "modalidade": "Corrida de Rua",
        "data": "01/02/2026",
        "local": "Parque Rui Ramos",
        "imagem": "https://via.placeholder.com/400x200",
        "site": "https://cronometraeventos.com.br",
        "instagram": "@minervarun",
        "historico": "Evento esportivo tradicional."
    },
    {
        "nome": "Pernas Solidárias",
        "modalidade": "Corrida Beneficente",
        "data": "26/04/2026",
        "local": "Alegrete - RS",
        "imagem": "https://via.placeholder.com/400x200",
        "site": "https://www.sesc-rs.com.br",
        "instagram": "@sescrs",
        "historico": "Corrida com arrecadação solidária."
    }
]

# Gerar JSON
with open("eventos.json", "w", encoding="utf-8") as arquivo:
    json.dump(eventos, arquivo, ensure_ascii=False, indent=4)

# Gerar CSV
with open("eventos.csv", "w", newline="", encoding="utf-8-sig") as arquivo:
    escritor = csv.DictWriter(arquivo, fieldnames=eventos[0].keys())
    escritor.writeheader()
    escritor.writerows(eventos)

print("=" * 50)
print("CATÁLOGO ESPORTIVO DE ALEGRETE")
print("=" * 50)
print("\nArquivos gerados com sucesso:")
print("✓ eventos.json")
print("✓ eventos.csv")
print("✓ index.html")
print("\nAbra o arquivo index.html para visualizar o catálogo.")

🧠 Reflexão sobre o Aprendizado

O que aprendi: Este projeto demonstrou como a programação Python aliada à Inteligência Artificial pode ser aplicada para organizar e valorizar o patrimônio cultural e esportivo de um município. A geração automática de arquivos em múltiplos formatos (JSON, CSV, HTML) permite que os dados sejam utilizados de forma versátil, desde planilhas até páginas web. O uso do ChatGPT para auxiliar na concepção e desenvolvimento da solução mostrou como as IAs podem acelerar o processo de criação de ferramentas úteis para a comunidade.
